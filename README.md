# TailWind Prac 
<br>

## Clone 
<pre>
git clone https://github.com/eomssy/tailwind-prac.git
cd tailwind-prac
</pre>

## Install 
<pre>
npm init 
npm install express 
npm install -D tailwindcss
npm install tailwindcss init
npx tailwindcss -i ./public/css/input.css -o ./dist/output.css --watch 
</pre>

## Run
<pre>
node index.js
</pre>
